help

This is a highly customizable minimalist launcher for Chrome, with a solely command-based UI.

Command List

  • help - This brings up this menu.
  • ? - This will search DuckDuckGo: "? apple" will search for apple. To change, see searchEngine
  • ! - This will go to a bookmark (including, crucially, JS bookmarklets)
  • bookmark - This will save a bookmark (and any JS code as a bookmrklet)
  • note - Brings up a temporary space to write a note whilst working. WARNING: these aren't saved!
  • map - Allows for single letter shortcuts e.g. "map r" & "! reddit" will jump to reddit when 'r' is pressed or "map a" & "? apple" maps 'a' to a search for "apple"
  • colour - Changes the accent colour of the app i.e. the loading bar, accepts standard colours, HEX and RGB
  • searchEngine - Change the search engine by passing the bit-before-the-query in the URL, say "https://www.bing.com/search?q=" for Bing.

JavaScript Coding

Javascript scripts can be executed, just like bookmarklets would be, but we take care of all of the handling Just type "js [filename]" where [filename] is the name of your desired script then type your script in the box To just execute some code, type "js" and then the code in the box. This will not be saved. To execute a saved script, type "!" and the name of the script.

Some helpful, built-in, functions

  • print(message, color [optional] ) - Displays a custom message in the standard toast box. Can only be 1 'print' per script, like return in most lnaguages.
  • DEFAULT_COLOUR - The default accent colour.
  • DEFAULT_SEARCH_ENGINE - The default search engine, DuckDuckGo.
